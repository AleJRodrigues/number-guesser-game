Number Guesser Game 🎮

A simple terminal-based game developed in Java where the player needs to guess a secret number between 1 and 100. This project was built to practice core programming logic, loops, and conditional statements.
🚀 Features

    Three difficulty levels: Easy (10 attempts), Medium (5 attempts), and Hard (3 attempts).

    Dynamic feedback: The game tells you if the secret number is higher or lower than your guess.

    Clean code structure using Java's modern switch expressions.

🛠️ Technologies Used

    Java 17+ (or your current version)

    java.util.Scanner for user input.

    java.util.Random for secret number generation.

🧠 What I Learned

    During the development of this project, I practiced:

    Encapsulation and method organization.

    Input handling and buffer clearing with input.nextLine().

    Control flow logic with for loops and if/else statements.

    Jogo de Adivinhação de Números 🎮

Um jogo simples para terminal desenvolvido em Java, onde o jogador deve adivinhar um número secreto entre 1 e 100. Este projeto foi criado para praticar lógica de programação, estruturas de repetição e condicionais.
🚀 Funcionalidades

    Três níveis de dificuldade: Fácil (10 tentativas), Médio (5 tentativas) e Difícil (3 tentativas).

    Feedback dinâmico: O jogo informa se o número secreto é maior ou menor que o seu palpite.

    Estrutura de código limpa utilizando as novas switch expressions do Java.

🛠️ Tecnologias Utilizadas

    Java 17+ (ou a versão que você estiver usando no UFBA)

    java.util.Scanner: Para entrada de dados do usuário.

    java.util.Random: Para geração do número aleatório.

🧠 Aprendizados

    Neste projeto, coloquei em prática:

    Organização de métodos e encapsulamento.

    Manipulação de entrada de dados e limpeza de buffer com input.nextLine().

    Lógica de controle de fluxo com laços for e condicionais if/else.
